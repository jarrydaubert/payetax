# PayeTax v4.9.5 Release Tweet

## Stats: v4.5.0 (Dec) → v4.9.5 (Jan 2026)

| Metric | Before | After |
|--------|--------|-------|
| Commits | - | +380 |
| Tests | ~2200 | 2,804 |
| Bundle | 15MB | 2.8MB (-81%) |
| Blog posts | ~10 | 25 |

## Tech Stack (bleeding edge)

- Next.js 16.1.4 + Turbopack
- React 19.2.3 (RC → stable)
- Tailwind CSS 4.1.18
- Zod 4.3.6 (`.pipe()` patterns)
- TypeScript 5.9.3 (strict)
- Biome 2.3.12 (replaced ESLint+Prettier)
- Bun 1.3.3 (replaced npm)
- Zustand 5 for state

## Performance Wins

- DeferredContent with IntersectionObserver (lazy calc)
- Dynamic imports for Recharts (-571KB initial)
- Server-rendered hero for instant LCP
- MutationObserver for lazy content scroll
- PWA with offline support

## New Systems

- Newsletter via Resend Audience + React Email
- Sentry error tracking → Linear auto-tickets
- 115 test suites, zero regressions
