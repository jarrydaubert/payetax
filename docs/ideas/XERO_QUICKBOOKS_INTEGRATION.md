## Why This Matters

This is the difference between "Calculator" and "Software".

Integration creates:
- Stickiness (users won't switch)
- Higher perceived value
- B2B sales opportunity
- Recurring revenue potential

## Features

### Phase 1: Export to Accounting
- Push dividend declaration to Xero/QBO
- Create journal entry for salary
- Record employer NI liability

### Phase 2: Import from Accounting
- Pull YTD salary/dividends paid
- Show remaining optimal extraction
- "Top up" recommendations

## Technical Requirements

- Xero API OAuth2 integration
- QuickBooks API OAuth2 integration
- Secure token storage
- Webhook handling for sync
