# SEO Technical Reviews

Purpose: monthly technical SEO checkpoint for indexability and sitemap quality.

## Workflow

1. Create current month report:
   - `bun run seo:report:init`
2. Complete checklist + actions:
   - `docs/reports/seo-tech/YYYY-MM.md`
3. Verify report exists:
   - `bun run seo:report:status`

## Focus Areas

- GSC coverage errors and trends
- Sitemap inclusion quality (salary + competitor subsets)
- Canonical consistency
- Structured data validation spot checks
